Sobre o FrontQuiz

O FrontQuiz é um website desenvolvido como projeto da disciplina de Programação Front-End. Seu objetivo é permitir que estudantes testem seus conhecimentos sobre HTML, CSS e JavaScript de forma simples e interativa.

O site foi desenvolvido utilizando HTML5 para estruturar as páginas, CSS3 para criar um layout agradável e responsivo, e JavaScript para adicionar interatividade ao quiz e ao formulário de contato.

Este projeto demonstra conceitos importantes do desenvolvimento Front-End, como navegação entre páginas, formulários, validação de dados, manipulação do DOM e design responsivo.

O FrontQuiz foi criado com foco na aprendizagem prática, permitindo que os usuários reforcem conceitos fundamentais do desenvolvimento Front-End por meio de um quiz simples e intuitivo.
